# Tienda Tech

## Descripción

Tienda Tech es un proyecto final de Front End que simula una tienda online de varios productos.

## Tecnologías utilizadas

- HTML
- CSS
- Bootstrap
- JavaScript
- Fetch API
- LocalStorage

## Funcionalidades

- Visualización de productos desde una API.
- Carrito de compras.
- Agregar y eliminar productos.
- Total de la compra.
- Persistencia del carrito con LocalStorage.
